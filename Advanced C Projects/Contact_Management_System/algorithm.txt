================================================================================
                        CONTACT MANAGEMENT ALGORITHM
================================================================================
=>Step 1: Initialize System
1.1 Set contactCount = 0
1.2 Set nextId = 1
1.3 Call loadContactsFromFile()


=>Step 2: Load Contacts from File (loadContactsFromFile)
2.1 TRY to open file "contacts.dat" in binary read mode ("rb")
2.2 IF file exists:
    2.2.1 Read contactCount from file (fread)
    2.2.2 Read all contacts (fread contacts array)
    2.2.3 Read nextId from file
    2.2.4 Close file
2.3 ELSE:
    2.3.1 Set contactCount = 0
    2.3.2 Set nextId = 1


=>Step 3: Save Contacts to File (saveContactsToFile)
3.1 Open file "contacts.dat" in binary write mode ("wb")
3.2 IF file opened successfully:
    3.2.1 Write contactCount to file (fwrite)
    3.2.2 Write entire contacts array (fwrite)
    3.2.3 Write nextId to file (fwrite)
    3.2.4 Close file
3.3 ELSE:
    Show error (silent failure)


=>Step 4: Main Menu Display (mainMenu)
4.1 REPEAT forever:
    4.1.1 Clear screen (system("cls"))
    4.1.2 Display menu header
    4.1.3 Display options 1-8
    4.1.4 Display total contacts count
    4.1.5 Get user choice (1-8)
    4.1.6 CALL appropriate function based on choice:
        choice=1 → addContact()
        choice=2 → viewAllContacts()
        choice=3 → searchContact()
        choice=4 → viewContactDetails()
        choice=5 → editContact()
        choice=6 → deleteContact()
        choice=7 → deleteAllContacts()
        choice=8 → BREAK loop and exit
    4.1.7 Wait for key press after each operation
4.2 Exit program


=>Step 5: Add New Contact (addContact)
5.1 Clear screen
5.2 Display "ADD NEW CONTACT" header
5.3 IF contactCount >= 500 THEN:
    Print "Contact list is full!"
    WAIT for key press
    RETURN
5.4 Create newContact structure
5.5 newContact.id = nextId
5.6 Increment nextId by 1
5.7 Prompt: "Enter Name: "
5.8 Read name using fgets (allow spaces)
5.9 Remove newline character from name
5.10 Prompt: "Enter Phone Number: "
5.11 Read phone using scanf
5.12 Prompt: "Enter Email: "
5.13 Read email using scanf
5.14 Prompt: "Enter Address: "
5.15 Read address using fgets (allow spaces)
5.16 Remove newline from address
5.17 Add to contacts[contactCount] = newContact
5.18 Increment contactCount by 1
5.19 CALL saveContactsToFile()
5.20 Display success message
5.21 WAIT for key press


=>Step 6: View All Contacts (viewAllContacts)
6.1 Clear screen
6.2 Display "ALL CONTACTS" header
6.3 IF contactCount == 0 THEN:
    Print "No contacts found!"
6.4 ELSE:
    6.4.1 Print table header with columns: ID, Name, Phone, Email
    6.4.2 FOR i = 0 to contactCount-1:
        Print contacts[i].id, name, phone, email in formatted columns
    6.4.3 Print total contacts count
6.5 WAIT for key press


=>Step 7: Search Contact (searchContact)
7.1 Clear screen
7.2 Display "SEARCH CONTACT" header
7.3 Display search options:
    1. Search by Name
    2. Search by Phone Number
7.4 Get user choice
7.5 IF choice == 1 (Search by Name):
    7.5.1 Prompt: "Enter Name: "
    7.5.2 Read searchName (allow partial matches)
    7.5.3 Print table header
    7.5.4 found = 0
    7.5.5 FOR i = 0 to contactCount-1:
        IF searchName appears anywhere in contacts[i].name:
            Print contact details
            found = 1
    7.5.6 IF found == 0:
        Print "No contact found with name: searchName"
7.6 ELSE IF choice == 2 (Search by Phone):
    7.6.1 Prompt: "Enter Phone Number: "
    7.6.2 Read searchPhone
    7.6.3 Print table header
    7.6.4 found = 0
    7.6.5 FOR i = 0 to contactCount-1:
        IF contacts[i].phone EXACTLY matches searchPhone:
            Print contact details
            found = 1
    7.6.6 IF found == 0:
        Print "No contact found with phone: searchPhone"
7.7 WAIT for key press


=>Step 8: View Contact Details (viewContactDetails)
8.1 Clear screen
8.2 Display "VIEW CONTACT DETAILS" header
8.3 Prompt: "Enter Contact ID: "
8.4 Read id
8.5 found = 0
8.6 FOR i = 0 to contactCount-1:
    IF contacts[i].id == id THEN:
        Print full contact details (ID, Name, Phone, Email, Address)
        found = 1
        BREAK
8.7 IF found == 0:
    Print "Contact not found!"
8.8 WAIT for key press


=>Step 9: Edit Contact (editContact)
9.1 Clear screen
9.2 Display "EDIT CONTACT" header
9.3 Prompt: "Enter Contact ID to edit: "
9.4 Read id
9.5 index = -1
9.6 FOR i = 0 to contactCount-1:
    IF contacts[i].id == id THEN:
        index = i
        BREAK
9.7 IF index == -1:
    Print "Contact not found!"
    WAIT for key press
    RETURN
9.8 Display current details:
    Name, Phone, Email, Address
9.9 Prompt for new details with current values shown in brackets:
    9.9.1 "Name [current]: " → if new name entered, update
    9.9.2 "Phone [current]: " → if new phone entered, update
    9.9.3 "Email [current]: " → if new email entered, update
    9.9.4 "Address [current]: " → if new address entered, update
9.10 CALL saveContactsToFile()
9.11 Display "CONTACT UPDATED SUCCESSFULLY!"
9.12 WAIT for key press


=>Step 10: Delete Contact (deleteContact)
10.1 Clear screen
10.2 Display "DELETE CONTACT" header
10.3 Prompt: "Enter Contact ID to delete: "
10.4 Read id
10.5 index = -1
10.6 FOR i = 0 to contactCount-1:
    IF contacts[i].id == id THEN:
        index = i
        BREAK
10.7 IF index == -1:
    Print "Contact not found!"
    WAIT for key press
    RETURN
10.8 Display contact found (Name, Phone)
10.9 Prompt confirmation: "Are you sure? (Y/N): "
10.10 Read confirm character
10.11 IF confirm == 'Y' OR confirm == 'y':
    10.11.1 FOR i = index to contactCount-2:
        contacts[i] = contacts[i+1]  // Shift contacts left
    10.11.2 Decrement contactCount by 1
    10.11.3 CALL saveContactsToFile()
    10.11.4 Display "CONTACT DELETED SUCCESSFULLY!"
10.12 ELSE:
    Print "Deletion cancelled."
10.13 WAIT for key press


=>Step 11: Delete All Contacts (deleteAllContacts)
11.1 Clear screen
11.2 Display "DELETE ALL CONTACTS" header
11.3 IF contactCount == 0:
    Print "No contacts to delete!"
    WAIT for key press
    RETURN

11.4 Display "Total Contacts: contactCount"
11.5 Display warning: "This will delete ALL contacts!"
11.6 Prompt: "Are you sure? (Y/N): "
11.7 Read confirm character
11.8 IF confirm == 'Y' OR confirm == 'y':
    11.8.1 Set contactCount = 0
    11.8.2 Set nextId = 1
    11.8.3 CALL saveContactsToFile()
    11.8.4 Display "ALL CONTACTS DELETED SUCCESSFULLY!"
11.9 ELSE:
    Print "Deletion cancelled."
11.10 WAIT for key press


=>Step 12: Exit System
12.1 Display "Thank you for using Contact Management System!"
12.2 Return from mainMenu()
12.3 End program (main returns 0)

================================================================================
                            END ALGORITHM
================================================================================

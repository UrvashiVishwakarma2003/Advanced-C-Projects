=>Step 1: Program Start
1.1 Include header files (stdio.h, string.h, stdlib.h)
1.2 Define structure "User" with members:
    - id (integer)
    - name (character array of size 50)
    - email (character array of size 50)
    - password (character array of size 50)
1.3 Declare function prototypes:
    - addUser()
    - viewUsers()
    - deleteUser()
    - loginUser()
    - changePassword()
    - userExists()


=>Step 2: Main Function Start
2.1 Declare integer variable 'choice'
2.2 Print welcome message
2.3 START infinite WHILE loop (while(1)):
    2.3.1 Display Main Menu:
          "1. Add New User"
          "2. View All Users"
          "3. Delete User"
          "4. User Login"
          "5. Change Password"
          "6. Exit"
    2.3.2 Print "Enter your choice: "
    2.3.3 Read integer input using scanf() and store in 'choice'
    2.3.4 SWITCH based on 'choice' value:
          CASE 1: Call addUser() function
          CASE 2: Call viewUsers() function
          CASE 3: Call deleteUser() function
          CASE 4: Call loginUser() function
          CASE 5: Call changePassword() function
          CASE 6: Print goodbye message and exit program (return 0)
          DEFAULT: Print "Invalid choice! Please enter 1-6 only."
    2.3.5 Go back to Step 2.3.1 (loop continues)


=>Step 3: addUser() FUNCTION
3.1 Declare variable 'u' of type struct User
3.2 Open file "users.txt" in append mode ("a")
3.3 IF file pointer is NULL THEN:
        Print error message "File open nahi ho raha!"
        RETURN from function
3.4 Print "========== ADD NEW USER =========="
3.5 Print "Enter User ID: "
3.6 Read integer input using scanf() and store in u.id
3.7 Call userExists(u.id) to check if ID already exists
3.8 IF userExists returns 1 THEN:
        Print "Error: User ID already exists!"
        Close file using fclose()
        RETURN from function
3.9 Print "Enter User Name: "
3.10 Read string input using scanf() and store in u.name
3.11 Print "Enter User Email: "
3.12 Read string input using scanf() and store in u.email
3.13 Print "Enter User Password: "
3.14 Read string input using scanf() and store in u.password
3.15 Write to file using fprintf(): "%d %s %s %s\n", u.id, u.name, u.email, u.password
3.16 Close file using fclose()
3.17 Print "User added successfully!"
3.18 Print user ID and Name
3.19 RETURN


=>Step 4: viewUsers() FUNCTION
4.1 Declare variable 'u' of type struct User
4.2 Open file "users.txt" in read mode ("r")
4.3 IF file pointer is NULL THEN:
        Print "No users found! Database empty hai."
        RETURN from function
4.4 Print table header: "========== ALL USERS LIST =========="
4.5 Print column headers with formatting
4.6 WHILE (fscanf() != EOF):
        Read user data: u.id, u.name, u.email, u.password
        Print user data in table format with proper spacing
4.7 Close file using fclose()
4.8 Print footer line
4.9 RETURN


=>Step 5: userExists(id) FUNCTION
INPUT: integer id
OUTPUT: 1 if user exists, 0 if not found
5.1 Declare variable 'u' of type struct User
5.2 Open file "users.txt" in read mode ("r")
5.3 IF file pointer is NULL THEN:
        RETURN 0 (file doesn't exist, so user not found)
5.4 WHILE (fscanf() != EOF):
        Read user data: u.id, u.name, u.email, u.password
        IF u.id == id THEN:
            Close file using fclose()
            RETURN 1 (user found)
5.5 Close file using fclose()
5.6 RETURN 0 (user not found)


=>Step 6: deleteUser() FUNCTION
6.1 Declare variables:
    - id (integer)
    - found = 0 (integer)
    - u (struct User)
6.2 Print "========== DELETE USER =========="
6.3 Print "Enter User ID to delete: "
6.4 Read integer input using scanf() and store in 'id'
6.5 Open file "users.txt" in read mode ("r")
6.6 IF file pointer is NULL THEN:
        Print "No users found!"
        RETURN from function
6.7 Open temporary file "temp.txt" in write mode ("w")
6.8 WHILE (fscanf(file) != EOF):
        Read user data: u.id, u.name, u.email, u.password
        IF u.id == id THEN:
            Set found = 1
            CONTINUE (skip writing this user to temp file)
        Write user data to temp file using fprintf()
6.9 Close both files using fclose()
6.10 Delete original file using remove("users.txt")
6.11 Rename temp file to "users.txt" using rename()
6.12 IF found == 1 THEN:
        Print "User with ID deleted successfully!"
     ELSE:
        Print "User with ID not found!"
6.13 RETURN


=>Step 7: loginUser() FUNCTION
7.1 Declare variables:
    - id (integer)
    - found = 0 (integer)
    - password[50] (character array)
    - u (struct User)
7.2 Print "========== USER LOGIN =========="
7.3 Print "Enter User ID: "
7.4 Read integer input using scanf() and store in 'id'
7.5 Print "Enter Password: "
7.6 Read string input using scanf() and store in 'password'
7.7 Open file "users.txt" in read mode ("r")
7.8 IF file pointer is NULL THEN:
        Print "No users registered yet!"
        RETURN from function
7.9 WHILE (fscanf(file) != EOF):
        Read user data: u.id, u.name, u.email, u.password
        IF u.id == id AND strcmp(u.password, password) == 0 THEN:
            Set found = 1
            BREAK out of loop
7.10 Close file using fclose()
7.11 IF found == 1 THEN:
         Print "LOGIN SUCCESSFUL!"
         Print "Welcome back, [name]!"
         Print "Email: [email]"
     ELSE:
         Print "LOGIN FAILED!"
         Print "Invalid User ID or Password!"
7.12 RETURN


=>Step 8: changePassword() FUNCTION
8.1 Declare variables:
    - id (integer)
    - found = 0 (integer)
    - oldPass[50], newPass[50] (character arrays)
    - u (struct User)
8.2 Print "========== CHANGE PASSWORD =========="
8.3 Print "Enter User ID: "
8.4 Read integer input using scanf() and store in 'id'
8.5 Print "Enter Old Password: "
8.6 Read string input using scanf() and store in 'oldPass'
8.7 Open file "users.txt" in read mode ("r")
8.8 Open temporary file "temp.txt" in write mode ("w")
8.9 IF file pointer is NULL THEN:
        Print "No users found!"
        RETURN from function
8.10 WHILE (fscanf(file) != EOF):
         Read user data: u.id, u.name, u.email, u.password
         IF u.id == id AND strcmp(u.password, oldPass) == 0 THEN:
              Set found = 1
              Print "Enter New Password: "
              Read string input using scanf() and store in 'newPass'
              Copy newPass to u.password using strcpy()
         Write user data to temp file using fprintf()
8.11 Close both files using fclose()
8.12 Delete original file using remove("users.txt")
8.13 Rename temp file to "users.txt" using rename()
8.14 IF found == 1 THEN:
         Print "Password changed successfully!"
     ELSE:
         Print "Invalid User ID or Password!"
8.15 RETURN


=>Step 9: Program Exit
